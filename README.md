## Overview

**SnakeGame.Az** is a clean, neon-themed version of the timeless Snake game, rendered on an HTML5 canvas with **no frameworks and no build step**. It plays smoothly on both desktop and touch devices.

## Features

- 🎨 Modern **neon visual style** on a crisp canvas.
- 📱 **Fully responsive** — keyboard on desktop, swipe/on-screen controls on mobile.
- 🏆 **Score tracking** with a persistent high score.
- ⚡ Smooth, frame-rate-independent game loop.
- 🪶 **Zero dependencies** — pure HTML/CSS/JS.

## Run Locally

```bash
git clone https://github.com/goshgarhasanov/snake-az.git
cd snake-az
python3 -m http.server 8000   # → http://localhost:8000
```

## Controls

- **Desktop:** arrow keys / WASD.
- **Mobile:** swipe or on-screen buttons.
